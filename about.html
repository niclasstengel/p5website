This website is a compilation of my p5 sketches from the course "Visual Stories with Code and Data" at the University of Applied Sciences Potsdam given by Lucas Vogel. The course aimed to provide an outlook on the potential of code for all types of designers. We learned the basics of programming with JavaScript and its code library p5.js. Additionally, we gained an insight into the basics of HTML and CSS to create this website, where we can now showcase, share, and collect our p5.js experiments.
